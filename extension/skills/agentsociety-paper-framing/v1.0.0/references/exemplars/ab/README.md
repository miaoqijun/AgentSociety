# PDF body text + figure assets

- Included: abstract, introduction/background, results/analysis, discussion/conclusion.
- Excluded: methods/materials and methods sections, references, acknowledgements, unrelated preceding/following article fragments, download watermarks.
- Figures are saved as rendered crops under `assets/<AB_id>/`.
