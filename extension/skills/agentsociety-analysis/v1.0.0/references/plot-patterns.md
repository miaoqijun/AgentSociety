# Plot Patterns

Compatibility alias for older references. The main pattern set now lives in:

- `references/common-patterns.md`
- `references/chart-types.md`

## How to Use It

- for multi-panel, legend, axis, or composition patterns, open `references/common-patterns.md`
- for chart-family selection first, open `references/chart-types.md`
- for panel-level assembly, combine those with `references/composite-figures.md`
